KVUC shared_script dokumentation

Introduktion

Når man vil gøre brug af “shared_funktions”, så er et overblik over de forskellige funktioner rigtig godt for hurtigt at benytte sig af disse. Til dette formål er der her lavet en demo-og-dokumentations-side (som kun eksisterer i work-flowet), meget lig den man finder i JQuerys dokumentation over deres funktioner. Ideen er at man får vist koden i funktionen og minimum et eksempel på anvendelsen af funktionen.

Hvordan

For at tilføje en beskrivelse/dokumentation af en funktion, følger man blot anvisningerne i den nedenstående demo.

Demo

Kopier nedenstående HTML kode og tilføj de nødvendige forklaringer. 

							<div class="Section">
									<h2 class="SectionHeader">my_shared_function()</h2>
									<p>
										<span class="Desc Where">Hvor: </span> Beskriv hvor funktionen anvendes..  <br/>
										<span class="Desc why">Hvorfor: </span> Beskriv behovet for denne funktion... <br/>
										<span class="Desc How">Hvordan: </span> En kort beskrivelse af hvordan funktionen virker... <br/>
										<span class="Desc Param">Argumenter: </span> Beskriv hvilke (om nogen) input/argumenter funktionen tager...  <br/>
										<span class="Desc Return">Output: </span> Beskriv retur data (om nogen) fra funktionen... <br/>
									</p>
									<pre class="brush: js;">
										my_shared_function() { 
										    Lorem ipsum dolor sit amet... 
										}
									</pre>
									<div class="DemoWrapper">
										<div class="DemoHeader"><span class="glyphicon glyphicon-chevron-right"></span> Demo</div>
										<div class="DemoContainer">
											<p> Lidt ekstra forklaringer om nødvendigt... </p>
											<div class="DemoCode"></div>  <!-- Koden fra nendenstående iframe indsættes automatisk i ".DemoCode" klassen -->
											<p> Lidt ekstra forklaringer om nødvendigt... </p>
											<iframe src="html/one_line_footer.html"></iframe>
										</div>
									</div>
							</div>   <!-- END: Section -->

Vigtigt: klassen ".DemoCode" er reserveret til automatisk indsætning af koden fra iframe-tag'et, som man kan vælge at tilføje hvis man ønsker at lave en demo på en seprat side.

For at tilføje koden fra en iframe, indsættes funktionen "ReturnAjaxData()" med funktionens navn i iframens "src" property. Her er der vigtig at navet på HTML-filen der skal iframes, er det samme som navnet der skrives i div-tag'et ".SectionHeader". Ved at navnet i div-taget ".SectionHeader" og filnavnet er ens, kan koden fra filen automatisk indsætes i tag'et med klassen ".DemoCode". 

							<script type="text/javascript">
									ReturnAjaxData("GET", "html/my_shared_function.html", false, "html"); 
							</script>

Man kan selv tilføje mere kildekode ved at indsætte det i et pre-tag således: 

 <pre class="brush: XXX;"> <!-- din kildekode her --> </pre>

- hvor XXX sættes lig "xml" for html (uden anførselstegn), "css" for CSS og "js" for JavaScript. Her er det dog bedst at anvende "&lt;" for < og "&gt;" for > på tags med de ovennævnte reserverede klasser og script-tags, idet der som nævnt fortages script-styrede handlinger på disse klasser og at scripts i script-tags ikke har noget JQuery eller inkluderede scripts at referere til.

Alle funktionsbeskrivelserne sorteres automatisk alfabetisk. Ligeledes genereres en "klikbar" liste automatisk over alle de beskrevne funktioner.

embedlink()

Hvor: Funktionen anvendes på såkaldte "visningssites" for de enkelte fag.
Hvorfor: Funktionen er nødvendig idet den automatisk laver indlejringskode til lærene til anvendelse i moodle og fronter
Hvordan: Funktionen har to komponenter: een til generering af link og en anden til styring fanerne til af valget mellem moodle og fronters indlejringskode.
Argumenter: Ingen
Output: Ingen. 

				        function embedlink(obj) {
								var UrlVarStr;
								UrlVarStr = String(window.location).split("/", 3).join("/").replace("http", "https");
								console.log("embedlink - UrlVarStr: " + UrlVarStr);

								var HrefObj = obj.parent().parent().find("a").eq(0).attr("href").replace("../", "/");
								console.log("embedlink - HrefObj: " + HrefObj);

								var embedFronter = '';
								var embedMoodle = '';

								var embedArray = [embedMoodle, embedFronter];

								var embedwrapping = "
Indsæt dette link i dit LMS eller på din webside
Fronter(embed)
Moodle(iframe)
Hjælp til indlejring (embedding) 
";

								var embedWidth;
								var embedHeight;


								// Klik på embedding knapper funktionalitet:

								//Hvis den man klikker på allerede har en parent...: 
								if (obj.parent().parent().find(".embedToggle").length > 0) {
								    $(".embedToggle").slideUp(150, function() {

								        $(".embedToggle").remove();
								        // Animation complete.
								    });

								} else {
								    if ($(".embedToggle").length > 0) {
								        // console.log("indeks: " + obj.parent().parent().index());
								        $(".embedToggle").slideUp(150, function() {

								            $(".embedToggle").remove();
								            // Animation complete.
								            obj.parent().parent().append(embedwrapping);
								            $(".embedToggle").slideUp(0);
								            $(".embedToggle").slideDown("slow");
								            $(".tab").click(function() {

								                var indeks = $(this).index();
								                $(".tab").removeClass("activetab");
								                $(this).addClass("activetab");
								                //alert (indeks);
								                changeLink(indeks);
								            });

								        });
								        //

								    } else {
								        //console.log("indeks: " + obj.parent().parent().index());
								        obj.parent().parent().append(embedwrapping);
								        $(".embedToggle").slideUp(0);
								        $(".embedToggle").slideDown("slow");
								        $(".tab").click(function() {

								            var indeks = $(this).index();
								            $(".tab").removeClass("activetab");
								            $(this).addClass("activetab");
								            //alert (indeks);
								            changeLink(indeks);
								        });
								    }
								}


								function changeLink(indeks) {

								    console.log("clickede på noget")

								    $(".embedtext").val(embedArray[indeks]);

								}

								$(".embedtext").click(function() {
								    $(this).select();
								});
						}
